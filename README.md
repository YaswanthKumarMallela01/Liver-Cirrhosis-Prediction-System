# Liver Cirrhosis Prediction System

This project is a complete end-to-end machine learning and Flask web application designed to predict the probability of liver cirrhosis based on patient clinical parameters.

---

## Project Overview
The objective of this system is to leverage data analytics and supervised learning to assist early-stage identification of liver cirrhosis risk. The workflow includes exploratory data analysis, model development, performance evaluation, and deployment through a Flask-based web interface.

---

## Key Features
- End-to-end data science and deployment structure
- Structured data analysis scripts
- Multiple model experimentation support
- Flask web interface for live predictions
- Modular folder organization
